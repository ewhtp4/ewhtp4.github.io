Skippy

A high performance but simple TCP/HTTP server written in C++ and using Qt5 Framework.

The idea behind Skippy is to have a backup Software that can have your site up and running in no time if your primary server is down.
Features: multithreading, directory privilege check and change, Site's Page view and extreme simplicity. No configuration needed. Tested on Debian/Ubuntu only.
